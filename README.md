# 🚀 Pankaj Connect - সম্পূর্ণ ফ্রি ক্লাউড ERP সফটওয়্যার

## বৈশিষ্ট্য

### ✅ 40+ Free APIs Integrated
1. **NID Services**
   - NID যাচাইকরণ
   - NID থেকে ছবি এক্সট্র্যাকশন
   - NID ডেটা এক্সট্র্যাক্টশন

2. **Location & Geolocation**
   - লাইভ জিপিএস লোকেশন
   - রিভার্স জিওকোডিং
   - ঠিকানা থেকে কোঅর্ডিনেট

3. **Communication**
   - এসএমএস সেন্ডিং
   - ইমেইল সেন্ডিং
   - বাল্ক কমিউনিকেশন

4. **Financial**
   - মুদ্রা রূপান্তর
   - পেমেন্ট গেটওয়ে ইন্টিগ্রেশন
   - ব্যাংক যাচাইকরণ

5. **Document Processing**
   - অক্টিক্যাল ক্যারেক্টার রিকগনিশন (OCR)
   - ডকুমেন্ট এক্সট্র্যাকশন
   - ডকুমেন্ট ভেরিফিকেশন

6. **Business Management**
   - ইনভয়েস ম্যানেজমেন্ট
   - ইনভেন্টরি ট্র্যাকিং
   - গ্রাহক ডাটাবেস
   - সাপ্লায়ার ম্যানেজমেন্ট

7. **HR & Payroll**
   - কর্মচারী ম্যানেজমেন্ট
   - উপস্থিতি ট্র্যাকিং
   - পেরোল সিস্টেম
   - ছুটির অনুরোধ

8. **Financial Reporting**
   - আয়-ব্যয় প্রতিবেদন
   - ব্যালেন্স শীট
   - লাভ-ক্ষতি বিবৃতি

### 🎨 প্রযুক্তি স্ট্যাক
- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Backend:** REST APIs
- **ডেটাবেস:** ক্লাউড-ভিত্তিক
- **নিরাপত্তা:** এনক্রিপশন সমর্থিত
- **ভাষা:** বাংলা ও ইংরেজি

- ✓ ট্যাবলেট
- ✓ মোবাইল ডিভাইস
- ✓ রেসপন্সিভ ডিজাইন

## মডিউল

### 1. ড্যাশবোর্ড
- রিয়েল-টাইম ডেটা ভিজুয়ালাইজেশন
- কী মেট্রিক্স
- সংক্ষিপ্ত সারাংশ

### 2. NID সেবা
- ডেটা যাচাইকরণ
- ছবি প্রসেসিং
- ডেটা এক্সট্রাকশন

### 3. লোকেশন ট্র্যাকিং
- রিয়েল-টাইম জিপিএস
- ট্র্যাকিং ইতিহাস
- ম্যাপ ভিজুয়ালাইজেশন

### 4. ইনভয়েস ব্যবস্থাপনা
- স্বয়ংক্রিয় ইনভয়েস জেনারেশন
- পেমেন্ট ট্র্যাকিং
- পিডিএফ এক্সপোর্ট

### 5. ইনভেন্টরি ম্যানেজমেন্ট
- স্টক ট্র্যাকিং
- কম স্টক সতর্কতা
- পণ্য বিভাগীকরণ

### 6. আর্থিক ব্যবস্থাপনা
- হিসাব লেজার
- আয়-ব্যয় ট্র্যাকিং
- লাভ-ক্ষতি হিসাব

### 7. কর্মচারী ম্যানেজমেন্ট
- কর্মচারী রেকর্ড
- পেরোল সিস্টেম
- উপস্থিতি ট্র্যাকিং

### 8. রিপোর্টিং
- বিক্রয় রিপোর্ট
- ইনভেন্টরি রিপোর্ট
- আর্থিক প্রতিবেদন

## ইনস্টলেশন

```bash
# ক্লোন করুন
git clone https://github.com/Pankaj52944/pankaj-connect.git

# ডিরেক্টরিতে যান
cd pankaj-connect

# সরাসরি index.html খুলুন বা একটি লোকাল সার্ভার চালান
python -m http.server 8000
```

## ব্যবহার

1. ব্রাউজারে `http://localhost:8000` খুলুন
2. যেকোনো মডিউল নির্বাচন করুন
3. ফর্মগুলি পূরণ করুন এবং জমা দিন
4. রিয়েল-টাইম ডেটা পান

## API কনফিগারেশন

সমস্ত API এন্ডপয়েন্ট `js/api-config.js` ফাইলে সংজ্ঞায়িত।

```javascript
const API_CONFIG = {
    nid: {
        verify: 'https://api.example.com/nid/verify',
        toImage: 'https://api.example.com/nid/to-image',
        extract: 'https://api.example.com/nid/extract-data',
    },
    // ... আরও 40+ APIs
};
```
